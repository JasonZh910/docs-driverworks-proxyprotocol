## has display

This capability specifies whether this device type has a video display. Its value is a Boolean string. For example:  “True” or “False”


### Signature

`<has_display></has_display>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <has_display>True</has_display>
</capabilities>
```
