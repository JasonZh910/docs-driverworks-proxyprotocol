## driver\_arch\_type

This capability is included in the SDK for Control4 internal use.


### Signature

`<driver_arch_type></driver_arch_type>`


| Parameter | Description |
| --- | --- |
| num | 0-5. Device Type Number |


### Example

```xml
<capabilities>
   <driver_arch_type>3</driver_arch_type>
</capabilities>
```
