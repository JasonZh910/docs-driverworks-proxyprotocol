## HEAT SETPOINT MAX F

Maximum Setpoint, int 0-100 that will do the same thing as the [`setpoint_heat_max`][1] capability.


### Signature

`HEAT_SETPOINT_MAX_F ()`


| Parameter | Description |
| --- | --- |
| int | Default 100 |


### Returns

`None`
