## HEAT SETPOINT MIN C

Minimum Setpoint, int 0-100 that will do the same thing as the [`setpoint_heat_min`][1] capability.


### Signature

`HEAT_SETPOINT_MIN_C ()`


| Parameter | Description |
| --- | --- |
| int | Default 100 |


### Returns

`None`
