## COOL SETPOINT MIN C


### Signature

`COOL_SETPOINT_ MIN_C ()`


| Parameter | Description |
| --- | --- |
| int | Minimum Setpoint, int 0-100 that will do the same thing as the [`setpoint_cool_min`][1] capability, default 100 |


### Returns

`None
`
