## SEND PGM COMMAND


### Signature

`SEND_PGM_COMMAND ()`


| Parameter | Description |
| --- | --- |
| int | PGM ID: Specify which zone is being affected. |
| str | COMMAND: Valid values: Open, Close and Toggle. |


### Returns

\`None


### Usage Note

PGM refers to ProGraMable input or output.
