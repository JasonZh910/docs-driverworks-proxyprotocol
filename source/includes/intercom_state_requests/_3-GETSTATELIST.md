## GET STATE LIST

This request is issued to obtain the device state of all of intercom endpoints in the project


### Signature

`C4:GET_STATE_LIST ()`


### Request Prototype

```
<state_list>
   <device_state id=”Id1”> 
     See GET_STATE response format
   </device_state>
   <device_state id=”Id2”> 
     See GET_STATE response format
   </device_state>
   <device_state id=”IdN”> 
     See GET_STATE response format
  </device_state>
</state_list>
```
