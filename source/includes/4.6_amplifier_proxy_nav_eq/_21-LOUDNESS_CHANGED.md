## LOUDNESS\_CHANGED

Navigator EQ notification that the loudness has changed.


### Signature

`LOUDNESS_CHANGED`


| Parameter | Type | Description                                           |
| --------- | ---- | ----------------------------------------------------- |
| OUTPUT    | int  | Output Binding ID                                     |
| LOUDNESS  | bool | Enabled True for loudness on, False for loudness off. |


### Returns

`None`
