## SET\_UP\_DOWN\_SWAP

Swap page UP/DOWN buttons (used by AVGen). `TRUE` to enable swap, `FALSE` to disable. 


### Name

`POUND ()`


| Parameter | Type | Description                      |
| --------- | ---- | -------------------------------- |
| SWAPFLAG  | BOOL | Ability to swap UP/DOWN buttons. |


### Returns

`None`
