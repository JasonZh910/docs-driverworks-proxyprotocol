## KEY PRESS

Command from the UI to the Proxy and then forwarded on to the Protocol. Has no Return values. Specifies that a key (button) was pressed on the UI interface. But different from`Â BUTTON_PRESS`, the actual key name is passed rather than its ID.



### Signature

`C4:KEY_PRESS()`


| Parameter | Description |
| --- | --- |
| str | InterfaceID:  A unique string to identify which interface is sending this command. |


### Returns

`None`
